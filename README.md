# D.R.E.A.M.S

**Deep Retention & Encoding AI Memory System.** Persistent, meaningful memory for the AI you work with.

Save your AI sessions with intention. Keep the *what*, the *where*, and the *why*, so the work,
the decisions, and the context carry forward instead of resetting every time the window closes.

> 🚧 **Public release coming soon.** This repository is being prepared for an open,
> run-it-yourself release: the full system with an empty database, so you bring your own
> memories. Watch the repo to follow along.
